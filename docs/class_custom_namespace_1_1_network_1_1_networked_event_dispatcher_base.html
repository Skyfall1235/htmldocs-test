Detailed Description

A generic base class for dispatching UnityEvents over the network using Unity Netcode for GameObjects.

Concrete implementations must override how the event is locally invoked and how it is broadcast to clients.

Template Parameters
TThe type of data carried by the event (e.g., string, int, Vector3).
