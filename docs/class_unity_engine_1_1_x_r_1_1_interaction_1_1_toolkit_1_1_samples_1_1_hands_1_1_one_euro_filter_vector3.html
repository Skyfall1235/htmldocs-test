Detailed Description

Provides a means to smooth jittery Vector3 signals. This filter is particularly effective for small and rapid movements, making it useful for applications like motion tracking or gesture recognition.

The filtering process relies on two main parameters: minCutoff and beta.

  • minCutoff primarily influences the smoothing at low speeds.
  • beta determines the filter's responsiveness to speed changes.

Initializes a new instance of the OneEuroFilterVector3 with specified cutoff and beta values.

Parameters
initialRawValueThe initial raw value for the filter.
minCutoffThe minimum cutoff value for the filter. Default is 0.1f.
betaThe beta value for the filter. Default is 0.02f.

Filter parameters:

  • minCutoff Controls the amount of smoothing at low speeds. A smaller value will introduce more smoothing and potential lag, helping to reduce low-frequency jitter. A larger value may feel more responsive but can let through more jitter. It's advised to start with a value around 0.1 for masking jitter in movements of about 1 cm.
  • beta Determines the filter's adjustment to speed changes. A smaller value provides consistent smoothing, while a larger one introduces more aggressive adjustments for speed changes, offering responsive filtering at high speeds. A starting value of 0.02 is recommended, but fine-tuning might be necessary based on specific use cases.
