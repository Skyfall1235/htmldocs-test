Detailed Description

A custom logging system for Unity, allowing for categorized and level-based logging with file output for errors and warnings. It provides customizable log settings via ScriptableObject, enabling developers to control which log categories are displayed in the console. Additionally, it includes utility functions for colorizing text and converting colors to hexadecimal strings.
