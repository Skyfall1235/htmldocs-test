Public Attributes

bool PreserveBorderEdges
 If the border edges should be preserved. Default value: false.
 
bool PreserveUVSeamEdges
 If the UV seam edges should be preserved. Default value: false.
 
bool PreserveUVFoldoverEdges
 If the UV foldover edges should be preserved. Default value: false.
 
bool PreserveSurfaceCurvature
 If the discrete curvature of the mesh surface be taken into account during simplification. Taking surface curvature into account can result in good quality mesh simplification, but it can slow the simplification process significantly. Default value: false.
 
bool EnableSmartLink
 If a feature for smarter vertex linking should be enabled, reducing artifacts in the decimated result at the cost of a slightly more expensive initialization by treating vertices at the same position as the same vertex while separating the attributes. Default value: true.
 
double VertexLinkDistance
 The maximum distance between two vertices in order to link them. Note that this value is only used if EnableSmartLink is true. Default value: double.Epsilon.
 
int MaxIterationCount
 The maximum iteration count. Higher number is more expensive but can bring you closer to your target quality. Sometimes a lower maximum count might be desired in order to lower the performance cost. Default value: 100.
 
double Agressiveness
 The agressiveness of the mesh simplification. Higher number equals higher quality, but more expensive to run. Default value: 7.0.
 
bool ManualUVComponentCount
 If a manual UV component count should be used (set by UVComponentCount), instead of the automatic detection. Default value: false.
 
int UVComponentCount
 The UV component count. The same UV component count will be used on all UV channels. Default value: 2.
