Detailed Description

Follow animation affordance for IPokeStateDataProvider, such as XRPokeFilter. Used to animate a pressed transform, such as a button to follow the poke position.

The Affordance System namespace and all associated classes have been deprecated. The existing affordance system will be moved, replaced and updated with a new interaction feedback system in a future version of XRI, including this sample script.
