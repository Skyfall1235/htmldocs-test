Detailed Description

Use this class to mediate the interactors for a controller under different interaction states and the input actions used by them.

If the teleport ray input is engaged, the Ray Interactor used for distant manipulation is disabled and the Ray Interactor used for teleportation is enabled. If the Ray Interactor is selecting and it is configured to allow for attach transform manipulation, all locomotion input actions are disabled (teleport ray, move, and turn controls) to prevent input collision with the manipulation inputs used by the ray interactor.
A typical hierarchy also includes an XR Interaction Group component to mediate between interactors. The interaction group ensures that the Direct and Ray Interactors cannot interact at the same time, with the Direct Interactor taking priority over the Ray Interactor.
