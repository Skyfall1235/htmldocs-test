Detailed Description

NetworkInteractableBase class synchronizes the <see cref=XRBaseInteractable"/> events over the network. Options are exposed to determine which functionality you want to syncrhonize.

This is meant to be a parent class that handles the core networking functionality. Classes can interhit from this class and override where applicable. See <see cref=NetworkPhysicsInteractable"/> for an example of how to extend this class.
